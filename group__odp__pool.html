◆ odp_pool_create()

odp_pool_t odp_pool_create ( const char *  name,
const odp_pool_param_t *  param 
)

Create a pool.

This routine is used to create a pool. The use of pool name is optional. Unique names are not required. However, odp_pool_lookup() returns only a single matching pool. Use odp_pool_param_init() to initialize parameters into their default values.

Parameters
nameName of the pool or NULL. Maximum string length is ODP_POOL_NAME_LEN.
paramPool parameters.
Returns
Handle of the created pool
Return values
ODP_POOL_INVALIDPool could not be created
Examples:
ipsec/odp_ipsec.c, ipsec_api/odp_ipsec.c, ipsec_offload/odp_ipsec_offload.c, odp_classifier.c, odp_generator.c, odp_ipfragreass.c, odp_l2fwd.c, odp_l2fwd_simple.c, odp_l3fwd.c, odp_ping.c, odp_pktio.c, odp_simple_pipeline.c, odp_switch.c, odp_timer_test.c, odp_traffic_mgmt.c, and time_global_test.c.

◆ odp_pool_destroy()

int odp_pool_destroy ( odp_pool_t  pool)

Destroy a pool previously created by odp_pool_create()

Parameters
poolHandle of the pool to be destroyed
Return values
0Success
-1Failure
Note
This routine destroys a previously created pool, and will destroy any internal shared memory objects associated with the pool. Results are undefined if an attempt is made to destroy a pool that contains allocated or otherwise active buffers.
Examples:
ipsec/odp_ipsec.c, ipsec_api/odp_ipsec.c, odp_classifier.c, odp_generator.c, odp_ipfragreass.c, odp_l2fwd.c, odp_l2fwd_simple.c, odp_l3fwd.c, odp_ping.c, odp_pktio.c, odp_simple_pipeline.c, odp_switch.c, odp_timer_test.c, odp_traffic_mgmt.c, and time_global_test.c.
