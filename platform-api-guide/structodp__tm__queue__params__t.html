Detailed Description

The odp_tm_queue_params_t record type is used to hold extra parameters when calling the odp_tm_queue_create() function.

Many of these fields are optional EXCEPT for priority. Also since it is expected that implementations might augment this record type with platform specific additional fields - it is required that odp_tm_queue_params_init() be called on variables of this type before any of the fields are filled in.

Examples:
odp_traffic_mgmt.c.
