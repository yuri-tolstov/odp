◆ hash_result_offset

uint32_t odp_crypto_packet_op_param_t::hash_result_offset

Offset from start of packet for hash result.

Specifies the offset where the hash result is to be stored. In case of decode sessions, input hash values will be read from this offset, and overwritten with hash results. If this offset lies within specified 'auth_range', implementation will mute this field before calculating the hash result.
