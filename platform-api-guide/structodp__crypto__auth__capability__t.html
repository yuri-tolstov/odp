odp_bool_t odp_crypto_auth_capability_t::bit_mode

Auth is operating in bitstring mode.

This auth works on series of bits, rather than sequences of bytes: auth_range in odp_crypto_op_param_t and odp_crypto_packet_op_param_t will use bits, rather than bytes.

Note: data buffer MUST start on the byte boundary, using offset which is not divisible by 8 is unsupported and will result in unpredictable behaviour.
